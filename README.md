# FUTURE_DS_02
Customer Retention &amp; Churn Analysis Dashboard built using Power BI. This project analyzes churn patterns, retention drivers, and customer lifetime trends to provide actionable business recommendations for improving customer retention in subscription-based businesses.


This project focuses on analyzing customer churn behavior using the Telco Customer Churn dataset. The objective is to identify churn patterns, high-risk customer segments, and key retention drivers that impact long-term customer value.

The dashboard provides insights into:

Overall churn rate and customer distribution

Churn by contract type, payment method, and internet service

Customer lifetime trends and tenure-based segmentation

Business-focused recommendations to reduce churn

This analysis simulates real-world SaaS retention analytics used by product and growth teams to improve customer lifetime value and reduce revenue loss.
